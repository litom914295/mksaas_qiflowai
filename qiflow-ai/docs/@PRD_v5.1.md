#### 3.3.3 传感器阈值与降级/拒答（v5.1 更新）
- 噪声阈值：磁力计 σ ≤ 2.5°（静态）/ ≤ 5°（动态）；陀螺仪漂移 ≤ 0.5°/min。
- 置信度档位：高（>0.9）/ 中（0.7–0.9）/ 低（<0.7）。低于 0.7 显示黄色/红色提示并触发校准引导；低于 0.4 拒答并转“手动输入”。
- 降级链路：全传感器 → 磁+加速度 → 仅磁 → 手动输入（地图/建筑对齐）。
- iOS 权限拒绝：强制手动输入 + 明示置信度低。

#### 3.4.2 积分消耗（v5.1 修订）
- 基础八字分析：10 积分（bazi=10）
- 风水分析：20 积分（xuankong=20）
- AI 对话（快速问答）：5 积分/次（aiChat=5）
- AI 深度解读（报告级）：30 积分/次（deepInterpretation=30）
- PDF 报告导出：5 积分（pdfExport=5）

#### 3.4.3 AI 服务计费明细（v5.1 新增）
- AI 对话（5）：短上下文、快速问答、成本可控；用于实时解疑与轻量建议。
- AI 深度解读（30）：长上下文、结构化报告、附证据引用；用于高价值决策支持。
- 订阅计划保留 lifetime（一次性激活终身权益），TECH 文档给出价格与权益口径并与 UI 展示对齐。

### 6.3 i18n 文化适配验收清单（v5.1 新增）
- 历法/时区/单位转换在英文与繁体环境可理解；提供“阳历/农历/节气”上下文辅助。

## 13. 数据与评测合同（v5.1 新增）

### 13.1 玄空（XuanKong）
- 基线：提供 24 山映射与兼线 ±3° 规则、九宫飞星公式的可测试定义。
- 黄金样例：≥300 个标注良好的标准用例（包含兼线/元运切换/飞星路径校验等边界）。
- 门槛：所有变更发布前需在黄金样例上“零回归回退”，并在 CI 中阻断未达标分支。

### 13.2 罗盘（Compass）
- 阈值：磁力计 σ 与陀螺仪漂移阈值如 3.3.3；置信度档位与 UI 三色联动（>0.9 绿、0.7–0.9 黄、<0.7 红）。
- 降级：<0.7 触发校准引导；<0.4 必须拒答并转“手动输入”流程；拒答需显示原因。
- 门槛：端到端“测量→降噪→置信度→UI 联动”通过自动化回归；CI 未达标阻断。

### 13.3 RAG（知识增强检索）
- 指标门槛（Golden Set 上）：
  - Recall@10 ≥ 85%
  - Faithfulness ≥ 95%
  - Answer Relevancy ≥ 90%
  - 拒答率（不可靠时拒答）≥ 95%
  - 引用可核性 100%（所有引用必须可追溯到真实知识源）
- 验收：在 CI 中计算上述指标，任何一项低于阈值即阻断发布（Week6/Week12 放行依赖通过）。
