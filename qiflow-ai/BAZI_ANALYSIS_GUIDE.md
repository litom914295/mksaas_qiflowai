# 🎯 QiFlow AI - 八字命理分析使用指南

## 📋 概述

QiFlow AI 现已集成业界领先的八字计算引擎，为用户提供专业级的命理分析服务。本指南将帮助您了解和使用这一强大的功能。

## 🚀 快速开始

### 方法1：通过主页访问

1. 启动应用：`npm run dev`
2. 打开浏览器访问主页
3. 点击页面上的 **"🎯 立即查看八字命理分析"** 按钮
4. 填写您的出生信息
5. 立即查看详细的八字分析报告

### 方法2：直接访问分析页面

1. 启动应用：`npm run dev`
2. 直接访问：`http://localhost:3000/zh-CN/bazi-analysis`
3. 按照页面指引填写信息并查看分析结果

## 📝 功能特性详解

### 🎨 界面特色

- **美观设计**：采用渐变背景和现代卡片布局
- **响应式布局**：完美适配桌面和移动设备
- **实时反馈**：加载状态和进度提示
- **多标签页**：丰富的分析内容分类展示

### 🔮 核心分析功能

#### 1. 基础八字分析

- **四柱信息**：年柱、月柱、日柱、时柱的完整信息
- **五行属性**：每个柱子的五行属性（金木水火土）
- **天干地支**：完整的干支组合显示

#### 2. 五行分析

- **五行分布图**：可视化展示五行强弱对比
- **平衡评估**：分析五行是否平衡
- **主导五行**：识别您命局中的主导元素
- **相生相克**：五行关系的详细分析

#### 3. 日主强度分析

- **强度等级**：强/弱/平衡的量化评估
- **分数显示**：0-100的强度评分
- **影响因素**：分析影响日主强度的关键因素
- **改善建议**：针对性的调理建议

#### 4. 大运分析

- **十年周期**：完整的十年大运周期
- **当前大运**：您当前所在的大运阶段
- **强度评估**：每个大运的运势强度
- **时间范围**：精确的大运起止时间

#### 5. 每日运势

- **综合评分**：今日运势的1-10评分
- **吉利活动**：适合今日进行的活动
- **不利活动**：今日应避免的活动
- **吉利方向**：今日适合的方位
- **个性化建议**：基于您八字的定制指导

#### 6. 互动分析

- **干支关系**：天干地支之间的互动
- **相生相克**：正向和负向的互动关系
- **强度评估**：互动关系的强弱程度
- **影响分析**：对整体命局的影响

- **详细的四柱信息**：每个柱子的完整属性
- **五行属性标识**：每个元素的颜色编码
- **互动关系展示**：柱子之间的关系分析

### 五行标签页

- **五行强度图表**：可视化的五行分布
- **平衡性分析**：五行是否和谐
- **个性化建议**：基于五行的调理建议

### 大运标签页

- **大运时间线**：十年为周期的运势分布
- **当前大运高亮**：您当前所在的大运阶段
- **运势强度指示**：每个大运的强度评估

- **运势评分**：今日运势的量化评分
- **活动建议**：吉利和不利活动的具体建议
- **方向指引**：今日适合的方位选择

### 洞察标签页

## 💡 使用建议

### 信息填写要点

1. **出生日期**：尽量选择公历，确保准确性
2. **出生时间**：如果记得确切时间，请填写
3. **时区选择**：根据出生地点选择正确的时区
4. **性别信息**：影响五行分析的准确性

### 分析结果应用

1. **日常决策**：参考每日运势进行选择
2. **重要时机**：把握大运的有利时期
3. **性格认识**：了解自己的五行特征
4. **长期规划**：基于大运周期做人生规划
