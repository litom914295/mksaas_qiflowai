# @UI_DESIGN_AI_BAZI_FENGSHUI_v5.1.1

风格：东方元素（八卦、金/赤/墨）+ 科技网格/霓虹/玻璃；留白充足，字号层级清晰，移动优先。

组件层级与间距（建议）
- 容器：max-w-screen-xl，x 24px，y 64~96px（首屏）
- 标题：text-4xl/5xl（移动/桌面），副标题 text-base/lg；强调用渐变或细发光
- 卡片：圆角 lg，1px 边，背景使用 2%~4% 透明度的玻璃化层
- 图标：本地 SVG（brand/*），线性渐变金/赤，大小 24/32/48

色彩（建议）
- 背景：#0B0C10 深夜蓝黑；
- 主色（点缀）：#D6A648（古金）、#E45D5D（朱红）、#58A6FF（科技蓝）；
- 文本：#F2F4F8 主、#A7B0BF 次；

可访问性
- 对比度 ≥ 4.5:1；交互元素有焦点状态；图片提供 alt；

排版
- Hero → Trust → Feature → HowItWorks → Teaser → Billing Hint → CTA → Compliance；
- 移动端首屏尽量在 1 屏内完成“承诺+主CTA+次CTA”；
