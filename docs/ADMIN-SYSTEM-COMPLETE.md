## 📈 关键指标说明

| 指标 | 说明 | 目标值 |
|-----|------|--------|
| K因子 | 每个用户带来的新用户数 | > 1.0 |
| 激活率 | 注册后完成激活的比例 | > 40% |
| 7日留存 | 7天后仍活跃的用户比例 | > 20% |
| 分享转化率 | 分享带来的新用户比例 | > 10% |
| 积分ROI | 积分投入产出比 | > 3.0 |

## 🛠️ 技术栈

- **前端**: Next.js 15, React 18, TypeScript
- **UI组件**: Shadcn/ui, Tailwind CSS
- **图表**: Recharts
- **数据管理**: React Query, Zustand
- **表单**: React Hook Form, Zod
- **认证**: Supabase Auth
- **数据库**: PostgreSQL (Supabase)

## 🐛 常见问题

### Q: 管理后台无法访问？
A: 检查是否使用管理员账户登录，清除浏览器缓存

### Q: 数据不显示？
A: 确认数据库连接正常，检查API响应

### Q: 图表加载慢？
A: 可能是数据量过大，尝试缩小时间范围
