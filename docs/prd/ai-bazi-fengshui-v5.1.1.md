# @PRD_AI_BAZI_FENGSHUI_v5.1.1

版本：v5.1.1（首页专项）
状态：草案（可执行）

一、项目背景与目标
- 通过“AI + 八字 + 风水罗盘”提供可信、专业、可落地的个人与空间解读，满足用户认知探索、趋利避害与改善空间的需求。
- 以首页为主战场，完成“认知建立 → 兴趣激发 → 功能试用 → 转化进入分析/咨询”的闭环。
- 视觉风格：东方传统（八卦、罗盘、金/赤/墨色调）+ 科技感（网格、微光、霓虹边、玻璃拟态），简洁大气。

二、用户细分与心理
- 新手型：希望快速得到“我是谁/近期趋势”的直观结果，偏好一键开始与示例。
- 求解型：有明确问题（工作、关系、搬迁、选址），在意“可信/专业/隐私”。
- 资深型：有基础，关注术语准确性、干货深度与付费价值。

三、核心价值（首页需体现）
- 专业可信：术语准确、方法论透明（不夸大、不捏造），专家团队把关与AI交叉验证。
- 个性化：基于生辰与坐标快速定制；可深度解读（十神/大运/流年，玄空飞星/纳气）。
- 隐私与合规：18+ 提示、免责声明、隐私与 DSAR 通道明确。
- 成本透明：计费口径清晰（aiChat=5、deepInterpretation=30、bazi=10、xuankong=20、pdfExport=5），余额不足三级降级策略。

四、信息架构（IA）
1) Hero（核心承诺与主CTA）
2) 信任背书（技术/专家/隐私）
3) 三大功能卡：八字分析｜风水罗盘｜AI 咨询（清晰入口）
4) How it works（3步即可）
5) 交互预览（罗盘微交互或示例结果）
6) 计费与权益提示（透明，避免意外付费压力）
7) CTA 区（再次引导）
8) 合规区（18+、免责声明、隐私与 DSAR）

五、关键文案（示例，可落地）
- 主标题：AI 八字风水 · 一键深度解读
- 副标题：融合东方智慧与现代 AI，让你更快看清当下与趋势。
- 主CTA：开始八字分析   次CTA：试用风水罗盘   辅助CTA：向 AI 提问
- 信任点：专业方法、AI 加持、隐私安全（三列简文案）
- 功能摘要：
  • 八字分析：一键生成命盘要点、十神关系与近期趋势。
  • 风水罗盘：基于坐标预览关键方位与飞星九宫提示。
  • AI 咨询：自然语言问答，获得针对性的建议与解释。
- 计费提示：分析前可见单次/项目消耗；余额不足将自动降级到简版结果。
- 免责声明：结果仅用于参考，不构成财务/医疗/法律等专业建议。

六、交互与状态（UX四态）
- Empty：无输入时展示示例与引导；
- Error：优先提示可自助修复（补全生辰、重试定位等）；
- Limited：余额不足→展示简版结果与充值入口；
- Timeout：请求超时→展示重试与离线导出提示；
- Suspense/Skeleton：关键区块提供骨架屏与流式加载。

七、非功能性与约束
- 性能：LCP 优先，首屏资源 < 200KB（不含字体与必要框架）。
- 安全与合规：18+，免责声明，隐私与 DSAR；不收集不必要数据。
- 国际化：保留 locale 结构；默认中文文案，允许后续接入翻译键。
- 不外链：不引用外部站点与素材；图片与图标使用本地或占位。

八、验收标准（关键）
- 三个CTA链接可达：/analysis/bazi、/analysis/xuankong、/ai/chat。
- “AgeVerification”和“DisclaimerBar”展示正常。
- 计费口径文案与 UI 提示到位，无误导或夸大。
- 四态组件可被切换渲染（可通过 prop/flag 触发）。
- 页面通过构建，无 ESLint/TS 明显错误。

九、依赖与路径
- 组件：src/components/qiflow/homepage/*
- 资源：public/brand/*（logo与图标占位）
- 入口页：src/app/[locale]/(marketing)/(home)/page.tsx

十、里程碑
- M1：PRD/任务生成
- M2：首页骨架上线（静态）
- M3：交互与文案打磨
- M4：可验证与A/B预留
