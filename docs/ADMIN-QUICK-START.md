# 🚀 QiFlow AI 管理后台快速启动指南

## ⚡ 快速开始（3分钟）

### 第一步：启动开发服务器

```bash
# 方式一：标准启动
npm run dev

# 方式二：快速启动（推荐）
npm run dev:fast
```

### 第二步：访问管理后台

打开浏览器，访问以下任一地址：

#### 选项 A：超级管理员登录（推荐）
```
http://localhost:3000/superadmin
```
- 专业的深色主题界面
- 安全提示和审计警告
- 自动跳转到管理后台

#### 选项 B：测试登录
```
http://localhost:3000/test-login
```
- 快速登录测试
- 自动识别用户类型

#### 选项 C：直接访问管理后台
```
http://localhost:3000/zh-CN/admin
```
- 需要先完成登录认证

### 第三步：使用管理员账户登录

| 账户 | 密码 |
|------|------|
| admin@qiflowai.com | Admin@123456 |

## 🔍 常见问题快速解决

### Q1: 无法启动开发服务器？
```bash
# 清理缓存并重新安装依赖
rm -rf .next node_modules
npm install
npm run dev
```

### Q2: 登录后显示权限不足？
**原因**: 使用了普通用户账户  
**解决**: 确保使用 `admin@qiflowai.com` 账户登录

### Q3: 管理后台页面显示404？
**检查项**:
1. 确认开发服务器正在运行
2. 检查URL是否正确（包含 `/zh-CN/` 前缀）
3. 尝试访问: `http://localhost:3000/zh-CN/admin`

### Q4: 数据不显示？
**可能原因**:
1. 数据库未正确连接
2. 没有测试数据
3. API 调用失败

**解决方案**:
```bash
# 检查数据库连接
npm run db:studio

# 添加测试数据（如果需要）
npm run seed:admin
npm run add-test-credits
```
