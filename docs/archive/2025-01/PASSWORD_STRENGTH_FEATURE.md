# 密码强度指示器功能 💪

## 📋 功能概述

已为注册表单添加了一个实时密码强度指示器，帮助用户创建更安全的密码。

## 🎨 功能特点

### 1. **实时强度评估**
密码强度分为 4 个等级：
- 🔴 **弱 (Weak)**: 1个进度条，红色
- 🟠 **一般 (Fair)**: 2个进度条，橙色
- 🟡 **良好 (Good)**: 3个进度条，黄色
- 🟢 **强 (Strong)**: 4个进度条，绿色

### 2. **智能建议**
根据密码不足之处，显示最多 2 条改进建议：
- ✓ 至少使用 8 个字符
- ✓ 添加小写字母 (a-z)
- ✓ 添加大写字母 (A-Z)
- ✓ 添加数字 (0-9)
- ✓ 添加特殊字符 (!@#$%^&*)

### 3. **评分算法**
```
总分 = 长度得分 + 字符类型得分

长度得分（最高 2 分）：
- 8+ 字符：+1 分
- 12+ 字符：+1 分

字符类型得分（每类 0.5 分）：
- 小写字母：+0.5 分
- 大写字母：+0.5 分
- 数字：+0.5 分
- 特殊字符：+0.5 分

强度等级：
- 0-1 分：弱
- 1.5-2.5 分：一般
- 3-3.5 分：良好
- 4 分：强
```

## 🌍 国际化支持

已支持以下语言：
- 🇬🇧 English (en)
- 🇨🇳 简体中文 (zh-CN)
- 🇯🇵 日本語 (ja)
- 🇰🇷 한국어 (ko)

## 💡 使用示例

### 密码示例与对应强度：

1. **`password`** → 🔴 弱
   - 建议：使用至少 8 个字符，添加大写字母

2. **`Password123`** → 🟠 一般
   - 建议：添加特殊字符

3. **`Password123!`** → 🟢 强
   - 所有条件都满足！

## 🎯 用户体验优势

✅ **减少注册摩擦**：单次密码输入 + 显示密码按钮
✅ **实时反馈**：用户边输入边看到强度变化
✅ **教育性**：通过建议帮助用户理解密码安全
✅ **视觉清晰**：颜色编码的进度条一目了然
✅ **无阻塞**：不强制要求强密码，但鼓励用户改进

## 🚀 技术实现

- **组件位置**: `src/components/auth/password-strength-indicator.tsx`
- **集成位置**: `src/components/auth/register-form.tsx`
- **翻译文件**: `messages/[locale].json` → `AuthPage.register.passwordStrength`
- **依赖**: React Hook Form 的 `useWatch` 钩子实时监听密码变化

## 🔧 配置选项

密码强度指示器支持以下 props：

```typescript
interface PasswordStrengthIndicatorProps {
  password: string;           // 必填：要评估的密码
  className?: string;         // 可选：自定义样式类
  showSuggestions?: boolean;  // 可选：是否显示建议（默认 true）
}
```
