## ❌ 如果发现问题

### 问题1: 仍然显示 `MISSING_MESSAGE` 错误
**解决方案**:
1. 确认已运行 `node update-homepage-i18n.js`
2. 重启开发服务器 (Ctrl+C，然后 `npm run dev`)
3. 清除浏览器缓存并刷新

### 问题2: 某个语言的翻译不正确
**解决方案**:
1. 编辑对应的语言文件: `src/locales/{locale}.json`
2. 找到对应的键值对并修改
3. 保存文件
4. 刷新浏览器

### 问题3: 部分UI元素没有翻译
**解决方案**:
1. 运行 `node verify-homepage-i18n.js` 检查翻译完整性
2. 如果缺少翻译键，编辑 `update-homepage-i18n.js` 添加
3. 重新运行 `node update-homepage-i18n.js`
4. 重启开发服务器
