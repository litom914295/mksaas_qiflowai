# AI风水大师完整使用指南

## 系统概述

AI风水大师是一个融合传统命理学与现代AI技术的智能咨询系统，通过"算法优先、语言后置"的架构设计，确保分析的准确性和专业性。

## 核心特性

### 1. 算法优先架构
- **精确计算**：所有命理和风水数据均由专业算法计算生成
- **数据驱动**：AI基于算法结果组织语言，避免幻觉和误导
- **可验证性**：计算结果可追溯，确保分析的科学性

### 2. 智能交互能力
- **自动识别**：智能识别用户意图和输入信息
- **友好引导**：缺少信息时提供清晰的补充指引
- **记忆上下文**：保持对话连续性，避免重复询问

### 3. 专业分析功能
- **八字命理**：四柱排盘、五行分析、十神定位、大运流年
- **玄空风水**：九宫飞星、方位吉凶、布局优化、流年调整
- **综合建议**：结合八字与风水提供个性化方案

- 使用绿色、木质装饰

- 太岁在东南方，不宜动土
- 岁破在西北方，需要化解

### 场景3：综合分析（八字+风水）

#### 用户输入
```
根据我的八字，如何调整家里的风水能提升财运？
```

#### 系统分析（已有八字和房屋数据）
```
根据您的八字命理和房屋风水综合分析：

【命理特点】
您的日主是癸水，五行水旺（45%），火弱（10%）
需要加强火元素来平衡命局，提升财运

【个性化风水调整方案】

🔥 南方离宫（火位）- 最重要
您的用神方位，必须重点强化：
• 设置为主要活动区域
• 大量使用红色、紫色装饰
• 安装暖色灯光，增加照明
• 摆放电器增强火气

💰 东北方财位强化
结合您的八字特点：
• 摆放红色聚宝盆（火生土旺财）
• 放置紫水晶洞（火色旺财）
• 使用暖黄色灯光
• 避免水景装饰

🛏️ 卧室优化建议
• 床头朝南最佳（补火）
• 使用红色、紫色、橙色床品
• 避免北方卧室（水位过旺）
• 床头柜放置小夜灯（增火气）

🎨 整体色彩方案
主色调：红色系、紫色系（30%）
辅助色：绿色系（木生火）（20%）
点缀色：黄色、橙色（15%）
避免色：黑色、深蓝色（水色）

这样的布局充分考虑了您八字缺火、水旺的特点，
结合房屋的九宫飞星吉凶方位，
能够最大化提升您的财运和整体运势。
```

## 常见问题

### Q1: 为什么需要精确的出生时间？
A: 八字命理以时辰为单位，两小时一个时辰，不同时辰出生的人命理格局完全不同。

### Q2: 房屋朝向如何测量？
A: 可以使用手机指南针APP，站在大门外面向外测量，得到的度数即为房屋朝向。

### Q3: AI的分析准确吗？
A: 系统采用"算法优先"原则，所有数据由专业算法计算，AI仅负责组织语言表达，确保了分析的准确性。

### Q4: 可以分析合婚吗？
A: 可以，提供双方的生辰八字信息即可进行合婚分析。

### Q5: 风水调整真的有用吗？
A: 风水是环境学的一部分，合理的布局确实能改善居住舒适度。但命运更多掌握在个人努力。
