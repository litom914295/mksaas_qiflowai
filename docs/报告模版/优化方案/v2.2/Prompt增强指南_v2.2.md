# Prompt 增强指南 v2.2

> 用于修改 `prompts/system/bazi_explainer.md` 和 `prompts/system/fengshui_consult.md`，使其输出符合 v2.2 报告要求。

---

## 核心原则

### 1. 语气与框架（8大用户心理需求对应）

| 用户心理需求 | Prompt 对应策略 |
|------------|----------------|
| **确定性焦虑** | 提供"权威外脑"式断言（如"根据50年命理数据库，您的格局属于..."）+ 信心度评分（如"85%确定性"） |
| **自我认知** | 生成"统一生命叙事"（如"您的人生主题：先蓄力、后爆发"），避免碎片化分析 |
| **掌控感** | 强调"可控因素"（如归因分解中"20%策略可调"），提供明确行动清单 + 时间窗口 |
| **归因需求** | 使用"4因素分解"（时间40% + 禀赋20% + 环境20% + 策略20%），标注哪些"不可控" vs "可控" |
| **决策支持** | 提供A/B/C对比 + 非最优方案的补救措施 + "即使选错也有补救"的安心话术 |
| **希望需求** | 明确"转折点时间"（如"2027年春季"）+ 量化改善（如"收入提升30-50%"）+ 3个"为什么会好"的理由 |
| **身份价值** | 使用VIP标记（如"专属分析"、"定制方案"、"180天跟踪"）+ 稀缺性（如"您的格局仅占人群15%"） |
| **行动指引** | 三级清单（必做/推荐/加分）+ 预期时效（如"1-2周见效"）+ 子步骤checklist |

### 2. 禁忌语言模式

| ❌ 避免 | ✅ 替换为 |
|--------|----------|
| "你能力不足" | "当前方法不匹配你的天赋" |
| "你八字不好" | "时机不利（占40%），非格局问题" |
| "命中注定失败" | "需调整策略，2027年后转机" |
| "你不适合这个" | "这个领域与你的用神相冲，建议调整切入角度" |
| "没办法改变" | "先天占20%，剩余80%可通过环境/策略优化" |
| "你运气差" | "当前大运不利（已完成35%），2027年转好" |

### 3. 数据可信度增强

- **引用来源**："根据《滴天髓》理论..." / "参考5000+案例数据库..."
- **证据链**："您18-25岁的挫折（对应忌神大运）→ 26-35岁积累期 → 36-45岁爆发期"
- **百分位**："您的格局纯度属于前15%"
- **历史对照**："类似格局的名人：某某（企业家）、某某（学者）"

## `bazi_explainer.md` 修改要点

### 增加模块：人生主题故事生成

```markdown
## 5. 生成人生主题故事（Life Theme）

你需要根据用户的格局、用神、大运，生成一个**统一的生命叙事**，包括：

### 5.1 主题标题
- 从格局推导主题类型：
  - 从格（顺势而为型）："随波逐流，顺势而为"
  - 用神受克（先蓄力后爆发型）："早年积累，中年爆发"
  - 大运前弱后强（逆袭型）："厚积薄发，晚年发力"
  - 格局纯粹（专精型）："专注一域，成为专家"

### 5.2 分阶段叙事
- 将用户人生分为3-5个阶段（每10年或每大运）
- 每个阶段包括：
  - **年龄范围**（如"18-25岁"）
  - **可能经历的事件**（如"求学压力大"、"职业探索"）
  - **命理意义**（如"用神未行运，需蓄力"）
  - **课题**（如"学会等待与积累"）
  - **习得能力**（如"抗压力"、"基础技能"）
  - **佐证**（如"大运为忌神"、"五行失衡"）

### 5.3 输出格式
```json
{
  "lifeTheme": {
    "title": "先蓄力、后爆发",
    "summary": "您的八字格局显示：早年需要积累，中年后运势提升...",
    "stages": [
      {
        "ageRange": "18-25岁",
        "likelyEvents": ["求学压力大", "职业探索期", "人际挫折"],
        "meaning": "用神未行运，需蓄力",
        "lesson": "学会等待与积累",
        "skills": ["抗压力", "基础技能"],
        "evidence": ["大运为忌神", "五行失衡"]
      }
    ]
  }
}
```
```

### 增加模块：归因分解（Attribution）

```markdown
## 6. 归因分解（Attribution Breakdown）

为了帮助用户"卸下自责"，你需要将当前困境/成就分解为4个因素：

1. **时间因素（Time）**：大运/流年的影响（通常占30-50%）
   - 示例："当前大运为忌神，占40%"
   - 标注："不可控，2027年后转好"

2. **先天禀赋（Endowment）**：八字格局本身（通常占10-30%）
   - 示例："格局中等，五行平衡，占20%"
   - 标注："不可控，但非破格"

3. **环境因素（Environment）**：风水、家庭、地域（通常占20-30%）
   - 示例："家居风水可优化，占20%"
   - 标注："可控，建议调整"

4. **策略因素（Strategy）**：职业选择、社交方式、方法论（通常占20-30%）
   - 示例："当前职业与用神不符，占20%"
   - 标注："可控，建议调整"

### 输出格式
```json
{
  "attribution": {
    "timeFactor": 40,
    "endowmentFactor": 20,
    "environmentFactor": 20,
    "strategyFactor": 20,
    "notes": [
      "时间因素（40%）：当前大运为忌神，2027年后转好",
      "先天禀赋（20%）：格局中等，非破格",
      "环境因素（20%）：家居风水可优化",
      "策略因素（20%）：职业/社交策略可调整"
    ]
  }
}
```

### 关键话术
- "您的当前困境主要源于时机不利（40%），而非能力不足"
- "先天因素只占20%，剩余80%可通过环境和策略改善"
- "这不是你不行，而是方法不匹配你的天赋"
```

## 8. 希望之光（Hope Timeline）

### 8.1 短期改善（6-12个月）
- 可能的小变化（如"人际改善"、"小额收入提升"）
- 关键事件时间（如"2025年春季"）

### 8.2 中期转折（1-3年）

### 8.4 三个"为什么会好"的理由

## `fengshui_consult.md` 修改要点

### 增加模块：可执行Checklist

```markdown
## 5. 可执行风水清单（Executable Checklist）

传统风水建议（如"东南宜水"）对普通用户太抽象，需转化为**具体任务**：

### 5.1 任务要素
- **宫位**（1-9）
- **八卦方位**（如"巽"、"乾"）
- **动作**（摆放/清理/移动/增亮/遮挡）
- **物品**（鱼缸/植物/柜子/灯具/镜子）
- **命理依据**（零正/飞星/用神）
- **优先级**（critical/major/minor）
- **预期影响**（如"提升财运10-15%"）
- **截止时间**（如"2025-02-15"）
- **周期**（weekly/monthly/quarterly，如需定期调整）

### 5.2 输出格式
```json
{
  "environmentChecklist": [
    {
      "id": "water-1",
      "palace": 1,
      "bagua": "坎",
      "task": "在1宫（客厅东北角）摆放鱼缸或流水摆件",
      "rationale": "该宫位为零神，见水旺财",
      "severity": "high",
      "expectedImpact": "提升财运10-15%",
      "dueBy": "2025-02-15",
      "recurrence": "quarterly"
    }
  ]
}
```

### 5.3 任务分类
- **水位任务**（零神宫位）：摆放鱼缸、饮水机、流水摆件、水培植物
- **山位任务**（正神宫位）：摆放高大植物、书柜、实心柜、泰山石
- **飞星任务**（五黄/二黑/病符）：挂铜葫芦、六帝钱、玉石
```

### 增加模块：零正审计（Zero-Positive Audit）

```markdown
## 6. 零正审计（Zero-Positive Audit）

检查当前布局是否"零正颠倒"（水山错位），并量化风险：

### 6.1 审计内容
- 是否"正神见水"（山位见水，损健康/事业）
- 是否"零神见山"（水位见山，损财运）
- 错位的宫位数量
- 风险等级（critical/major/minor/none）

### 6.2 输出格式
```json
{
  "zeroPositiveAudit": {
    "isReversed": true,
    "issues": [
      "6宫（正神位）摆放了鱼缸，建议移除",
      "1宫（零神位）堆放杂物，建议清理并摆水"
    ],
    "severity": "major"
  }
}
```

### 6.3 话术
- "您当前布局存在2处零正颠倒，可能导致财运受阻（占20%环境因素）"
- "建议在2025年春节前调整，预计财运可提升10-15%"
```

## 通用话术库（复制粘贴用）

### 确定性话术
- "根据《滴天髓》理论，您的格局属于..."
- "参考5000+命理案例，85%的情况下..."
- "您的格局纯度属于前15%（稀缺）"

### 自我认知话术
- "您的人生主题：先蓄力、后爆发"
- "您18-25岁的挫折并非偶然，而是命理预设的积累期"
- "您的核心优势在于..."

### 掌控感话术
- "以下3项行动，您可以立即开始"
- "1-2周内可见效"
- "最佳启动时间：2027年2-4月"

### 归因话术
- "时机占40%（不可控），策略占20%（可控）"
- "这不是你不行，而是方法不匹配"
- "先天因素只占20%，剩余80%可改善"

### 决策支持话术
- "方案A > 方案B，但若选B，可通过以下方式补救"
- "即使选错，也有补救窗口"
- "您有2次试错机会（2027年春、2028年秋）"

### 希望话术
- "2027年2月是关键转折点（还有18个月）"
- "届时收入可提升30-50%"
- "您的格局属于晚发型，中年后爆发力强"

### VIP话术
- "本报告为您定制，分析耗时48小时"
- "含12张专属图表"
- "附180天跟踪服务"
- "您的格局仅占人群15%（稀缺）"

### 行动指引话术
- "必做项（1-2周）：..."
- "推荐项（1-2月）：..."
- "加分项（3-6月）：..."
- "子步骤：1️⃣ ... 2️⃣ ... 3️⃣ ..."
