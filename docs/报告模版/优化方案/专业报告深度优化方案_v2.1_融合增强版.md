3. **第3大需求升级**: 控制感 + "我在主动掌控命运"

   - 心理洞察: 不是"我不行"，而是"方法不匹配"

5. **第5大需求升级**: 问题归因 + "卸下自责"
   - ✨ 新增: 多因素拆解表（时间/五行/环境/策略）
   - ✨ 新增: 责任分解（可控vs不可控）

6. **第6大需求升级**: 希望获得 + "心理续命卡"
   - ✨ 新增: 专门的"希望之光"章节

7. **第7大需求**: 身份认同 + "我值得被高级对待"（保留并增强）

   - ✨ 新增: 显性化投入成本说明
   - 心理洞察: 高价服务本身就是身份信号

8. **第8大需求**: 可执行的行动清单 🆕 全新！
   - ✨ 新增: 本周3件小事（降低门槛）

## 💡 核心话术模板库（新增3个）

### 模板1: 权威背书式决策建议 🆕

```markdown
### 关于您纠结的{decision_type}

从您的描述中，我能感觉到您内心其实已经有倾向了：
> "您更想选择{option_a}，对吗？"

从命理角度看，**您的直觉是对的**：
- {option_a}更符合您的{pattern_characteristic}特点
- 长期来看，{option_a}能最大化您的{advantage}优势

**如果您最终还是选了{option_b}**（迫于压力）：
- 这不是世界末日
- 但您需要做这些调整：{remedies}

> **最后的话**：跟随内心，少留遗憾。
```

### 模板2: 苦难意义化叙事 🆕

```markdown
### 关于您经历的{difficult_experience}

从命理角度看，这段经历并不是"命运的惩罚"，而是"命局的必修课"：

**它带给您什么？**
虽然痛苦，但这段经历培养了您的：
1. {skill_1}（这是35岁后爆发的关键能力）
2. {skill_2}（别人花钱学不到的"软实力"）

> **记住**：那些杀不死您的，终将使您更强大。
```

### 模板3: 从"我不行"到"方法不匹配" 🆕

```markdown
### 您不是不行，只是{current_method}不适合您

很多时候，我们会陷入"自我否定"的陷阱。
但从命理角度看，**真相是**：

✅ **不是您有问题，而是方法不匹配**

您的命局特点是{pattern_characteristic}：
- 适合{matched_approach}的做法
- 不适合{unmatched_approach}的做法

> **记住**：您只需要成为"最匹配自己命局"的版本。
```
