# 个人中心使用指南

## 欢迎使用QiFlow AI个人中心

个人中心是您管理账户、查看分析记录、管理积分等功能的核心枢纽。本指南将帮助您快速了解和使用各项功能。

## 快速开始

### 访问个人中心

1. 登录您的QiFlow AI账户
2. 点击页面右上角的用户头像
3. 选择"个人中心"进入

或直接访问：`https://your-domain.com/zh-CN/(personal)/dashboard`

## 功能模块详解

### 1. 仪表板 Dashboard

仪表板是您的个人中心首页，提供账户概览和快速操作入口。

#### 主要功能：

- **数据统计卡片**
  - 积分余额：显示当前可用积分
  - 分析次数：本月已使用的分析次数
  - 连续签到：当前连续签到天数
  - 邀请人数：成功邀请的用户数量

- **快速操作**
  - 快速分析：一键进入分析页面
  - 充值积分：快速充值积分
  - 每日签到：完成今日签到
  - 邀请好友：生成邀请链接

- **活动时间线**
  - 查看最近的操作记录
  - 了解账户动态

### 2. 我的分析 My Analysis

管理您的所有分析记录，支持搜索、筛选和导出。

#### 核心功能：

**查看分析记录**
- 按时间顺序展示所有分析
- 显示分析类型、标题、时间等信息
- 支持预览分析结果

**搜索与筛选**
- 搜索框：输入关键词搜索分析记录
- 类型筛选：按八字、罗盘、风水、姓名分类
- 收藏筛选：只显示收藏的分析
- 排序方式：按时间或名称排序

**管理操作**
- ⭐ 收藏：标记重要分析
- 👁️ 查看：查看完整分析结果
- 📥 导出：支持JSON、CSV、PDF格式
- 🗑️ 删除：删除不需要的记录

#### 使用技巧：
- 使用收藏功能标记重要分析，方便后续查找
- 定期导出重要分析结果备份
- 使用搜索功能快速定位特定分析

### 3. 积分管理 Credits

查看积分余额、充值和消费记录。

#### 主要功能：

**积分余额**
- 实时显示当前积分
- 查看积分有效期
- 了解积分获取途径

**充值套餐**
- 多种套餐选择
- 优惠活动展示
- 支持多种支付方式

**交易记录**
- 详细的收支明细
- 支持时间范围筛选
- 导出交易记录

**积分获取指南**
- 每日签到获取积分
- 邀请好友奖励
- 完成任务赚积分
- 参与活动获得额外积分

### 4. 个人资料 Profile

坚持签到，获得积分奖励。

## 常见问题 FAQ

### Q1: 如何快速找到之前的分析记录？

**答：** 使用"我的分析"页面的搜索功能：
1. 输入分析标题关键词
2. 选择分析类型筛选
3. 使用日期范围缩小范围
4. 查看收藏的分析

### Q2: 积分有什么用途？

**答：** 积分可用于：
- 进行各类分析（八字、风水、罗盘等）
- 解锁高级功能
- 兑换优惠券
- 参与特殊活动

### Q3: 如何提高账户安全性？

**答：** 建议您：
1. 设置强密码（包含大小写字母、数字、特殊字符）
2. 启用两步验证
3. 定期检查登录历史
4. 不要分享账户信息
5. 使用官方渠道登录

### Q4: 邀请好友有什么奖励？

**答：** 邀请奖励包括：
- 每成功邀请一位用户获得基础积分
- 被邀请用户首次充值，获得额外奖励
- 月度邀请榜前三名有特殊奖励
- 累计邀请达到一定数量解锁VIP特权

### Q5: 分析记录会保存多久？

**答：** 
- 免费用户：保存最近30天的记录
- VIP用户：永久保存所有记录
- 建议定期导出重要分析结果

### 提高效率的小技巧

1. **使用快捷键**
   - `Ctrl/Cmd + K`：快速搜索
   - `Ctrl/Cmd + /`：显示快捷键列表

2. **个性化设置**
   - 自定义仪表板显示内容
   - 设置常用功能快速访问
   - 调整通知偏好

3. **数据管理**
   - 定期清理不需要的分析记录
   - 使用标签分类管理
   - 导出重要数据备份

4. **积分优化**
   - 参与每日签到不间断
   - 关注优惠活动批量充值
   - 完成任务获取额外积分
