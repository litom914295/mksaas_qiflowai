# 🚀 快速开始 - 登录管理后台

## 方法一：使用快速启动脚本（推荐）

### Windows PowerShell

```powershell
# 在项目根目录运行
.\scripts\quick-start.ps1
```

脚本会自动：
1. ✅ 检查 Node.js 环境
2. ✅ 安装项目依赖
3. ✅ 配置环境变量
4. ✅ 启动开发服务器

## 方法二：手动启动

### 1. 安装依赖

```bash
# 使用 pnpm (推荐)
pnpm install

# 或使用 npm
npm install
```

### 2. 启动开发服务器

```bash
# 使用 pnpm
pnpm dev

# 或使用 npm
npm run dev
```

### 3. 访问管理后台

打开浏览器，访问：
- 🔐 **登录页面**: http://localhost:3000/zh-CN/admin/login
- 🏠 **管理首页**: http://localhost:3000/zh-CN/admin/dashboard
- 📚 **文档中心**: http://localhost:3000/zh-CN/admin/docs

注意：项目使用国际化路由，需要在URL中包含语言代码（zh-CN）

## 🔧 常见问题

### Q: 无法启动服务器？
**A**: 确保已安装 Node.js 20+ 和项目依赖
```bash
node --version  # 检查 Node.js 版本
pnpm install   # 重新安装依赖
```

### Q: 登录失败？
**A**: 
1. 检查邮箱和密码是否正确
2. 清除浏览器缓存
3. 使用演示账号快速登录按钮

### Q: 页面显示404？
**A**: 
1. 确认服务器已正常启动
2. 检查URL是否正确
3. 尝试访问 http://localhost:3000

### Q: 数据库连接失败？
**A**: 
1. 本地开发模式不需要数据库即可运行（使用模拟数据）
2. 如需连接真实数据库，请配置 `.env.local` 中的 `DATABASE_URL`
