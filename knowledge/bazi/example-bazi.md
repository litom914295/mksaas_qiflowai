# 八字基础知识

## 十神系统

### 食神

食神是八字命理学中十神之一，代表着一个人的才华、创造力和享受生活的能力。

食神的特性：
1. 代表才华横溢，艺术天赋
2. 性格温和，不喜争斗
3. 享受生活，注重品质
4. 有口福，善于品鉴美食
5. 财源之神，能生财

食神在不同位置的含义：
- 年柱食神：祖上富裕，童年幸福
- 月柱食神：父母疼爱，家庭和睦
- 日柱食神：配偶贤惠，婚姻美满
- 时柱食神：子女孝顺，晚年安康

### 伤官

伤官也是十神之一，代表反叛、创新和突破。

伤官的特点：
1. 聪明机智，反应敏捷
2. 个性张扬，不受约束
3. 创新能力强，敢于突破
4. 口才出众，善于表达
5. 容易与人发生冲突

## 五行生克

五行相生：
- 木生火：木材燃烧产生火
- 火生土：火烧成灰化为土
- 土生金：土中蕴藏金属矿物
- 金生水：金属熔化成液体
- 水生木：水滋养树木生长

五行相克：
- 木克土：树根破土而出
- 土克水：土能阻挡水流
- 水克火：水能灭火
- 火克金：火能熔化金属
- 金克木：金属工具砍伐树木
