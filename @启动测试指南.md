## 🐛 常见问题排查

### 问题1: 端口被占用
```bash
# 错误信息: Port 3000 is already in use
# 解决方案:
taskkill /F /IM node.exe
# 或更换端口:
$env:PORT=3001; npm run dev
```

### 问题2: 页面白屏
1. 打开浏览器控制台 (F12)
2. 查看 Console 和 Network 标签
3. 检查是否有错误信息
4. 尝试硬刷新 (Ctrl+Shift+R)

### 问题3: 组件不显示
1. 确认路由正确
2. 检查浏览器控制台错误
3. 尝试清除缓存重启
