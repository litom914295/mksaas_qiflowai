## 零正理论详解

### 什么是零正理论

零正理论是玄空风水中的核心理论之一，由蒋大鸿等玄空派大师所传承。**零神**和**正神**是根据三元九运确定的两个重要星神，决定了山水布局的吉凶。

**核心原则**：
- **零神主动主水**：零神宜动不宜静，宜见水不宜见山
- **正神主静主山**：正神宜静不宜动，宜见山不宜见水
- **零正不可颠倒**：颠倒则凶，轻则破财损丁，重则败家遭祸

### 零正神的确定方法

零正神的确定基于三元九运和"颠倒五行"原理：

#### 三元九运零正神对照表

| 元运 | 时间 | 正神 | 零神 | 说明 |
|------|------|------|------|------|
| **一运** | 1864-1883 | 1白 | 7赤 | 一白为正神，七赤为零神 |
| **二运** | 1884-1903 | 2黑 | 8白 | 二黑为正神，八白为零神 |
| **三运** | 1904-1923 | 3碧 | 9紫 | 三碧为正神，九紫为零神 |
| **四运** | 1924-1943 | 4绿 | 1白 | 四绿为正神，一白为零神 |
| **五运** | 1944-1963 | 5黄 | 2黑 | 五黄为正神，二黑为零神 |
| **六运** | 1964-1983 | 6白 | 3碧 | 六白为正神，三碧为零神 |
| **七运** | 1984-2003 | 7赤 | 4绿 | 七赤为正神，四绿为零神 |
| **八运** | 2004-2023 | 8白 | 5黄 | 八白为正神，五黄为零神 |
| **九运** | 2024-2043 | 9紫 | 6白 | 九紫为正神，六白为零神 |

#### 确定规律

**正神**：当运之星即为正神（当运星数 = 正神星数）

**零神**：正神对宫之星为零神
- 计算公式：零神 = 10 - 正神
- 一运正神1，零神 = 10 - 1 = 9，但实际环形循环为7（1的对宫）
- 八运正神8，零神 = 10 - 8 = 2，但实际为5（8的对宫）

**实际规律**（洛书九宫对宫关系）：
- 1-9对宫（但实际用1-7、4-1循环）
- 2-8对宫（实际用5-2、8-5循环）
- 3-7对宫（实际用9-6、6-3循环）
- 4-6对宫
- 5在中宫，以其三合局论

### 零正颠倒的危害

#### 零神见山（凶）

**零神宜动，见山则静**，违背了零神的属性：

**危害表现**：
- 财运受阻，经营困难
- 破财损耗，投资失败
- 生意萧条，客流稀少
- 财源不通，收入锐减

**严重程度**：
- **Minor（轻度）**：单个零神位见小山体
- **Major（中度）**：多个零神位见山或单个见大山
- **Critical（严重）**：零神位全部见山且正神也颠倒

#### 正神见水（凶）

**正神宜静，见水则动**，违背了正神的属性：

**危害表现**：
- 人丁不旺，家道衰落
- 散财败家，财帛难聚
- 事业不稳，频繁变动
- 健康问题，疾病缠身

**严重程度**：
- **Minor（轻度）**：单个正神位见小水体
- **Major（中度）**：多个正神位见水或单个见大水
- **Critical（严重）**：正神位全部见水且零神也颠倒

#### 双重颠倒（大凶）

**零神见山且正神见水**，双重违反原则：

**危害表现**：
- 财丁两败，家运衰落
- 诸事不顺，厄运连连
- 家破人亡，刑伤克损
- 宅运凶险，急需化解

### 正确的零正布局

#### 零神位布局（宜见水）

**适宜布置**：
1. **真水**：
   - 鱼缸、水族箱
   - 风水轮、流水摆件
   - 喷泉、水景
   - 加湿器

2. **动态元素**：
   - 电视、音响
   - 空调、风扇
   - 活动区域、走道
   - 门窗、出入口

3. **颜色配置**：
   - 蓝色、黑色系
   - 流动纹理装饰
   - 波浪、水纹图案

**布置要求**：
- 水体要有情环抱，不宜直冲
- 水质清洁，流动有序
- 大小适中，不宜过大过小
- 保持卫生，定期清理

#### 正神位布局（宜见山）

**适宜布置**：
1. **静物摆设**：
   - 高柜、书架
   - 衣柜、储物柜
   - 假山、石头
   - 厚实墙体

2. **稳定元素**：
   - 靠山沙发
   - 床头靠墙
   - 办公桌背靠
   - 重物摆放

3. **颜色配置**：
   - 黄色、褐色系
   - 土元素装饰
   - 稳重厚实图案

**布置要求**：
- 山体要端正庄严，不宜破碎
- 靠山稳固，给人安全感
- 高度适中，不宜过高压顶
- 避免尖锐，形态圆润为佳

## 函数使用示例

### 1. 获取零正神信息

```typescript
import { getLingzhengInfo } from '@/lib/qiflow/xuankong/lingzheng';

// 获取八运零正神信息
const info = getLingzhengInfo(8);

console.log('当运：', info.period); // 8
console.log('正神：', info.positiveGod); // 8
console.log('零神：', info.zeroGod); // 5
console.log('说明：', info.description); // "八运：八白为正神，五黄为零神"
```

**输出结构**：

```typescript
interface LingzhengPeriodInfo {
  period: Yun;              // 运数（1-9）
  positiveGod: FlyingStar;  // 正神星数
  zeroGod: FlyingStar;      // 零神星数
  description: string;      // 文字说明
}
```

### 2. 分析零正神位置

```typescript
import { analyzeLingzhengPositions } from '@/lib/qiflow/xuankong/lingzheng';

// 示例：八运飞星盘
const plate: Plate = [
  { palace: 1, periodStar: 8, mountainStar: 1, facingStar: 1 },
  { palace: 2, periodStar: 2, mountainStar: 3, facingStar: 3 },
  { palace: 3, periodStar: 5, mountainStar: 4, facingStar: 4 },
  // ... 其他6个宫位

const positions = analyzeLingzhengPositions(plate, 8);

console.log('零神位置：', positions.zeroGodPositions); // [3, 6, 9] - 震、乾、巽宫
console.log('正神位置：', positions.positiveGodPositions); // [1, 4, 7] - 离、巽、艮宫
console.log('零神详情：', positions.zeroGodDetails);
// 输出：[
//   { palace: 3, bagua: '震', starType: 'period' },
//   { palace: 6, bagua: '乾', starType: 'period' },
//   ...
// ]
```

**返回结构**：

```typescript
{
  zeroGodPositions: PalaceIndex[];     // 零神宫位数组
  positiveGodPositions: PalaceIndex[]; // 正神宫位数组
  zeroGodDetails: {                    // 零神详细信息

    bagua: string;

  }[];

```

### 3. 检查零正颠倒

```typescript
import { checkZeroPositiveReversed } from '@/lib/qiflow/xuankong/lingzheng';

const result = checkZeroPositiveReversed(
  plate,
  8,
  '子',
  '午',
  {
    waterPositions: [1, 4],   // 离宫、巽宫有水（正神位）
    mountainPositions: [3, 6] // 震宫、乾宫有山（零神位）
  }
);

console.log('是否颠倒：', result.isReversed); // true
console.log('严重程度：', result.severity);   // 'critical'
console.log('问题列表：', result.issues);
// 输出：[
//   "零神见山：震、乾宫零神见山，主破财损丁",
//   "正神见水：离、巽宫正神见水，主散财败家"
// ]
```

**严重程度分类**：

```typescript
type Severity = 'critical' | 'major' | 'minor' | 'none';

// critical: 零神见山且正神见水（双重颠倒）
// major: 零神见山或正神见水（单一颠倒）
// minor: 轻微不协调（保留扩展用）
// none: 无颠倒或缺乏环境信息
```

   - 错误：放置大鱼缸（水）
   - 违反：正神见水，主败家

4. **艮宫（正神位）**：
   - 错误：设置洗手间（水）
