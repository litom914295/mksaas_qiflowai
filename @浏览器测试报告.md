## 🐛 常见问题排查

### 问题1: 页面打不开
**症状**: 浏览器显示"无法访问此网站"  
**排查**:
```powershell
# 检查服务器是否运行
Get-Job | Where-Object {$_.State -eq 'Running'}

# 检查端口是否被占用
netstat -ano | findstr :3000

# 重启服务器
Stop-Job -Id 1
Start-Job -ScriptBlock { Set-Location D:\test\mksaas_qiflowai; npm run dev }
```

### 问题2: 点击宫位无反应
**排查**:
1. F12 打开控制台
2. 查看Console标签是否有错误
3. 检查Network标签确认资源加载
4. 尝试硬刷新 Ctrl+Shift+R

### 问题3: 详情面板不显示
**排查**:
1. 检查控制台是否有组件错误
2. 确认点击的元素正确
3. 查看是否有CSS样式冲突
4. 检查数据是否正确加载
